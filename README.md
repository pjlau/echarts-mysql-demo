# ECharts MySQL Demo

This project demonstrates a simple web application using ECharts for visualization, MySQL for data storage, FastAPI as the backend, and HTML for the frontend.

## Prerequisites
- Python 3.8+
- MySQL Server
- Node.js (optional, for local development server)
- Modern web browser
